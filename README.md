# 软件工程项目——中国象棋

华中科技大小2022届软件工程项目——中国象棋，前端部分，后端项目地址https://github.com/leicd458/My-CChess-Back

## 框架

Vue + Vite

## 如何运行

### 下载项目

```
git clone https://github.com/leicd458/My-CChess-Front.git
```

### 下载依赖

在项目的根目录下，输入

```
npm install
```

### 运行项目

```
npm run dev
```

### 打包成dist文件夹

```
npm run build
```
